 # K-Pop Group Page
A simple and interactive K-Pop Group webpage created using HTML, CSS, and JavaScript. This project was built as my first JavaScript task while learning the basics of DOM manipulation, specifically the getElementById() method.

## Project Overview
This webpage showcases a K-Pop group and demonstrates how JavaScript can be used to interact with HTML elements. The project helped me understand the fundamentals of selecting elements and updating webpage content dynamically.

## Features
Attractive K-Pop themed webpage
Built with HTML, CSS, and JavaScript
Uses JavaScript's getElementById() method
Beginner-friendly project

## Technologies Used
HTML5
CSS3
JavaScript
## What I Learned

Through this project, I learned:
How to connect JavaScript with HTML.
How to select elements using document.getElementById().
Writing and executing simple JavaScript code.
Structuring a small web development project.
