# K-Pop Groups Website

An interactive and responsive K-Pop groups website built using *HTML, **Tailwind CSS, and **JavaScript DOM manipulation*. Users can explore different K-Pop groups, view members, and switch between multiple themes.


##  About the Project

This project was created to practice JavaScript DOM concepts such as:

- document.getElementById()
- innerHTML
- src
- style.display
- Event handling using onclick

The website displays information about popular K-Pop groups and their members. Users can switch between groups and view individual member information dynamically without reloading the page.



##  Features

###  Group Information
- BTS
- BLACKPINK
- TXT
- Stray Kids
- Katseye
- More groups can be added easily.

###  Member Profiles
- Click on member images.
- Displays:
  - Member name
  - Member description
  - Large profile image

###  Multiple Themes
Users can switch between:

- 🌙 Dark Theme
- ☀️ Light Theme
- 🌸 Pink Theme
- 💜 Purple Theme

The website background and cards change dynamically.

###  Responsive Design
Built with Tailwind CSS to provide a responsive experience across:

- Desktop
- Tablet
- Mobile devices

###  Animations
- Hover effects
- Smooth transitions
- Image scaling effects
